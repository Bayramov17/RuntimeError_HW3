Recent Projects

Coding does not have any borders, and everyone can code! As a RuntimeError team of IT students, we did our first steps towards the galaxy of coding, and now we want to provide you a small journey of our projects.

Homework 2

Hour Of Code

“Everyone Can Program!”. Under this slogan every year, Hour Of Code creates a one-hour project to inspire kids to code. Our #RuntimeError group in 15.12.2020 made a one hour of meeting with 6th and 7th-grade students of Republican Humanities Gymnasium named after S.J.Pishevari, and observed methods of easy coding. It was a surprise for us to see such talented and creative kids who desire to learn programming. We hope that we gave them the first step to an amazing world of IT!

Homework 3

Creating a Website

Teamed up with the knowledge and a bit of minimalism in a design, we created an outstanding web-page, that has a big number of features and technologies implemented, such as HTML, CSS, JavaScript. Another important note to make is that the web-page is responsive, meaning it can be visited via mobile or tablet and the user will not lose the quality of the web-flow we tried to create. Advanced features are added, such as the "hamburger" navigation bar for mobile users and Dark Mode that keeps the web-experience the same even at night time. With the help of flexbox and CSS3, we were able to create smooth transitions and animations to improve the user experience.

Homework 4

"Robotics"

“Robotics” project was all about exploring how to build and write software for Lego Mindstorms robots. This project required us to use the special software EV3 Classroom. Having this software and the documentation from the Lego website, we were able to create software that allows for a robot to move along the line and avoid obstacles. The development environment of EV3 shares the same ideas as in Hour Of Code, which had a positive impact since we were already familiar with the structure. We won’t include specific details here, because the software is still in the stage of development, but we hope that we can achieve our goal as soon as possible.
